## Repository Overview

This repository contains **two independent projects** developed for the **Mobile Computing (CPM) course**, second semester, first year of MEIC Master's program at the Faculty of Engineering, University of Porto.

### Projects

1. **Project 1 – Smart Supermarket Environment**  
   Integrated systems for a smart supermarket: backend server, customer app, checkout terminal app, and QR code generator app.  
   [→ View Project 1 README](project_1/README.md)

2. **Project 2 – Weather Forecast Application**  
   Client-server weather application using Flutter, showing forecasts for Portuguese district capitals and allowing favorite cities management.  
   [→ View Project 2 README](project_2/README.md)
